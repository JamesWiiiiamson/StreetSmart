# Custom Marker Images

Place your custom marker images in this directory.

## Recommended Image Specifications:
- **Format**: PNG (with transparency)
- **Size**: 32x32px to 64x64px (recommended: 48x48px)
- **Anchor Point**: Bottom center (for proper pin placement)
- **Background**: Transparent

## Suggested File Names:

### Report Types:
- `bad-lighting.png` - For bad lighting reports
- `no-sidewalk.png` - For no sidewalk reports
- `suspicious-area.png` - For suspicious area reports
- `blocked-path.png` - For blocked path reports

### Crime Markers:
- `crime-high.png` - High severity crime
- `crime-medium.png` - Medium severity crime
- `crime-low.png` - Low severity crime

### Lighting Markers:
- `lighting-high.png` - High lighting level
- `lighting-medium.png` - Medium lighting level
- `lighting-low.png` - Low lighting level

### Other:
- `user-location.png` - User's current location
- `default-marker.png` - Default marker

## Where to Find Free Icons:

1. **Flaticon** - https://www.flaticon.com
   - Search for: "map pin", "location marker", "warning", "lightbulb", etc.
   - Download as PNG, 48x48px or 64x64px

2. **Icons8** - https://icons8.com
   - Search for map-related icons
   - Free with attribution

3. **Material Icons** - https://fonts.google.com/icons
   - Search: "place", "warning", "lightbulb", "construction"

4. **The Noun Project** - https://thenounproject.com
   - Search for safety and location icons

## Tips:
- Use consistent styling across all markers
- Keep file sizes small (< 10KB per image)
- Test markers at different zoom levels
- Consider using SVG for scalability (convert to PNG if needed)
